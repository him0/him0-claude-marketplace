# Character Skills Plugin

Claude Codeにキャラクター性を追加するスキルプラグインです。3種類の個性的なキャラクターとして応答できます。

## 特徴

- **技術的正確性を維持**: キャラクター性を持たせつつ、技術情報は正確に提供
- **分割ファイル構造**: キャラクター定義を独立したファイルに分離し、保守性を向上
- **柔軟な設計**: 新しいキャラクターの追加が容易

## 利用可能なキャラクター

### 1. 海賊 (Pirate) - `/pirate`

**キャラクター名**: キャプテン・ジャック

豪快で自由奔放な海賊キャラクター。冒険心あふれる口調で、海や航海に関連する比喩を使いながら技術サポートを提供します。

**使用例**:
```
/pirate
```

**口調の特徴**:
- 「〜じゃ」「〜だぜ」などの海賊らしい語尾
- 「ヨーホー!」「アホイ!」などの掛け声
- 航海や宝探しの比喩表現

### 2. 執事 (Butler) - `/butler`

**キャラクター名**: セバスチャン

礼儀正しく品格のある執事キャラクター。丁寧で洗練された言葉遣いで、プロフェッショナルなサポートを提供します。

**使用例**:
```
/butler
```

**口調の特徴**:
- 丁寧語・尊敬語の使用
- 「かしこまりました」「お任せください」などの表現
- 「〜でございます」といった品のある語尾

### 3. 博士 (Professor) - `/professor`

**キャラクター名**: ドクター・アインシュタイン

知識豊富で教育熱心な博士キャラクター。学究的でありながら親しみやすい口調で、詳しい解説とともにサポートを提供します。

**使用例**:
```
/professor
```

**口調の特徴**:
- 「〜じゃよ」「〜なのじゃ」といった老学者風の語尾
- 「ふむふむ」「なるほど」などの相槌
- 「興味深い!」「実に面白い!」などの感嘆表現

## ファイル構造

```
character-skills-plugin/
├── plugin.json                 # プラグイン定義
├── README.md                   # このファイル
├── commands/                   # コマンド定義
│   ├── pirate.md              # 海賊コマンド
│   ├── butler.md              # 執事コマンド
│   └── professor.md           # 博士コマンド
└── characters/                 # キャラクター詳細定義
    ├── pirate.md              # 海賊キャラクター詳細
    ├── butler.md              # 執事キャラクター詳細
    └── professor.md           # 博士キャラクター詳細
```

## 技術詳細

### ファイル読み込み機構

各コマンドファイル（`commands/*.md`）は、対応するキャラクター定義ファイル（`characters/*.md`）を以下の構文で読み込みます：

```markdown
{{READ:../characters/キャラクター名.md}}
```

この仕組みにより：
- キャラクター定義の再利用が可能
- キャラクター詳細の更新が容易
- コマンドとキャラクター定義の責任分離

### 新しいキャラクターの追加方法

1. `characters/` ディレクトリに新しいキャラクター定義ファイルを作成
2. `commands/` ディレクトリに対応するコマンドファイルを作成
3. `plugin.json` にコマンドを登録

## 注意事項

- キャラクター性は応答に個性を加えますが、技術的な正確性は常に維持されます
- コードやコマンドは明確に提示されます
- 深刻なエラーやセキュリティ問題には、適切なトーンで対応します
